# 電影檢索系統

## 簡介
建立一個可以查詢電影相關資料的網站，可由國家、導演、工作室、獎項、演員、評分……去搜尋出各個電影的資訊，除此之外還能夠進行新增及刪除，並透過圖形化的介面讓使用者更方便管理系統。
<br>
我們還有製作登入系統，只有root能夠新增與刪除資料，一般使用者則無
<br>

這個系統可以根據 :
1. 電影
2. 演員
3. 導演
4. 工作室
5. 評論
6. Casting
7. 獎項
來進行查詢、新增及刪除
